## 反馈

[发送反馈](https://github.com/1217950746/Smile-Theme/issues/new)

**图标适配原则：** 外形不会变更，内容会按照扩展名本身相关内容设计

**界面主题原则：** 只做 Bug 修复，与跟随 VSCode 版本更新

### 注意事项

要我适配图标的时候，需要提供相关的内容，网站、图片，都可以

反馈界面 Bug 的时候，需要提供 Bug 截图
